# **Age-Related Aberrant Splicing and Its Role in Elderly Breast Cancer Prognosis**

## Overview

This repository contains the data analysis for **Age-Related Aberrant Splicing and Its Role in Elderly Breast Cancer Prognosis**. It includes statistical analysis and visualizations of bioinformatics data, survival analyses, and genomic alterations.

## Requirements

- R software version 4.2.0
- Required R packages: survminer, complexheatmap, upsetR, oncoprint, cor.test, venn.diagram

### Installation

To install the required R packages, run the following in R:
install.packages("survminer")
install.packages("complexheatmap")
install.packages("upsetR")
install.packages("oncoprint")
install.packages("venn.diagram")
